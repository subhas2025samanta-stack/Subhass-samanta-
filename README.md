# Amar Business 3D Product Viewer

এটি একটি ইন্টারেক্টিভ থ্রিডি প্রোডাক্ট ভিউয়ার ওয়েবসাইট যা Three.js ব্যবহার করে তৈরি।

## ফিচার

✨ **থ্রিডি অবজেক্ট ডিসপ্লে** - সুন্দর থ্রিডি কিউব যা ইন্টারেক্টিভভাবে ঘোরানো যায়

🖱️ **মাউস কন্ট্রোল** - মাউস দিয়ে অবজেক্ট ঘোরান, জুম করুন এবং প্যান করুন

📱 **রেসপন্সিভ ডিজাইন** - সব ডিভাইসে পারফেক্টলি কাজ করে (মোবাইল, ট্যাবলেট, ডেস্কটপ)

🎨 **আধুনিক UI** - বাংলা ইন্টারফেস সহ স্টাইলিশ ডিজাইন

⚙️ **অপটিমাইজড পারফরমেন্স** - মসৃণ অ্যানিমেশন এবং দ্রুত লোডিং

## ফাইল স্ট্রাকচার

```
.
├── index.html          # মেইন HTML ফাইল
├── css/
│   └── style.css       # স্টাইলশীট
├── js/
│   └── main.js         # Three.js মূল স্ক্রিপ্ট
├── README.md           # ডকুমেন্টেশন
├── package.json        # প্রজেক্ট ম্যাটাডেটা
└── .gitignore          # Git ইগনোর ফাইল
```

## ইউজেজ

1. **লোকালি রান করুন:**
   ```bash
   # কোনো বিল্ড টুল প্রয়োজন নেই - শুধু index.html ওপেন করুন
   # অথবা লাইভ সার্ভার ব্যবহার করুন (ওয়েব মডিউল ইমপোর্টের জন্য)
   python -m http.server 8000
   # তারপর ব্রাউজারে যান: http://localhost:8000
   ```

2. **কন্ট্রোল:**
   - 🖱️ **বাম বাটন ড্র্যাগ**: থ্রিডি অবজেক্ট ঘোরান
   - 🖱️ **মধ্য বাটন/স্ক্রল**: জুম ইন/আউট
   - 🖱️ **ডান বাটন ড্র্যাগ**: প্যান
   - 🔄 **Reset View**: প্রাথমিক অবস্থানে ফিরে যান
   - 🎬 **Animation**: অটো রোটেশন টগল করুন
